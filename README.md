# SmartSpend: Personal Finance Management & Spending Prediction System

## Project Overview
SmartSpend is a web-based personal finance management application that helps users track income, expenses, budgets, savings goals, and financial reports. The system also includes a machine learning component that predicts end-of-month spending and supports smart financial recommendations.

## Objective
The goal of this project is to build an intelligent finance management system that combines:
- Transaction tracking
- Budget management
- Reports and charts
- Savings goals
- Alerts
- Smart financial recommendations
- Machine learning-based spending prediction

## Main Features
- User authentication
- Dashboard with income, expenses, and balance overview
- Transaction management
- Category-based budget tracking
- Financial reports and charts
- Savings goals tracking
- Alerts for budget limits
- Smart Advisor with recommendations
- End-of-month spending prediction using Linear Regression

## Machine Learning Component
The project integrates a Linear Regression model to predict the user’s expected total spending by the end of the month.

The model uses features such as:
- Average daily spending
- Number of transactions
- Previous month spending

The prediction is used by the Smart Advisor to evaluate financial risk and generate recommendations.
